# chatgpt-web-admin
基于Chatgpt-web项目，具有管理用户后台及支付卡密等功能


## 说明
> **此项目 Fork 自 [Chanzhaoyu/chatgpt-web](https://github.com/Chanzhaoyu/chatgpt-web), 新增了部分功能:**

## 开源计划
计划于5月开源体验版，完整商业版视情况而定。

## 已实现功能
[✓] 登录&注册&用户管理

[✓] 多客户端同步历史会话

[✓] 自定义或使用百度敏感词

[✓] 支付卡密

[✓] 用户消耗记录

[✓] ChatGpt绘画

## 短期计划
